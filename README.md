# 倉管系統 (Phase 1–3)

## 啟動方式

需要 Python 3.10+。資料庫檔：`warehouse.db`（SQLite，整個倉管資料都在這個檔，備份直接複製它）。
要重置資料：關掉程式 → 刪掉 `warehouse.db` → 再啟動。

### Windows
雙擊 `run.bat` 即可。第一次會自動建立虛擬環境並安裝套件。
啟動後開瀏覽器到 <http://127.0.0.1:8000>

### Linux / macOS server
```bash
git clone https://github.com/alien18331/warehouse_mgr.git
cd warehouse_mgr
chmod +x setup.sh
./setup.sh                       # 安裝 + 啟動 (預設 0.0.0.0:8000)
# 或分開:
./setup.sh install               # 只安裝
HOST=0.0.0.0 PORT=8080 ./setup.sh run   # 自訂 host/port 啟動
```

背景常駐建議用 systemd 或 `nohup ./setup.sh run > app.log 2>&1 &`。

## 已實作（Phase 1–3）

**主檔**：品牌、供應商、人員、存放位置、工號/案件、料件（可標 SN 追蹤、安全庫存）

**進貨單**（三種類型）：
- 新竹採購（帶 PO）
- 辦公室請購（直接綁工號）
- 餘料退回（從某張出貨單退回，自動標記 `is_surplus`）

**出貨單**（兩種類型）：
- 正常專案出貨
- 餘料轉撥出貨（只能從 `is_surplus=1` 庫存扣帳）

**庫存看板**：依品牌/型號/位置/餘料分組顯示，自動依 `inbound_lines − outbound_lines` 即時計算（`stock_balance` view）。低於安全庫存會在儀表板紅字提醒。

**序號追蹤**：可在進/出貨單明細逐個輸入 SN，狀態自動切換 `in_stock / shipped / returned`。SN 詳情頁可看完整履歷（哪張進貨單進、哪張出貨單出、目前在誰手上）。

**借出管理**：勁傑借出單號可登錄狀態（借出中 → 已歸還 → 已銷帳）。

## 操作流程示範

1. 先到「料件」新增你的料件（若是變頻器類請勾「逐 SN 追蹤」）。
2. 「進貨」→ 新竹採購 → 填日期、供應商、PO、明細 → 儲存。
3. 「出貨」→ 專案出貨 → 選工號 → 明細時下方會即時顯示該料目前各位置的庫存。
4. 出貨後若客戶退料 → 「進貨」→ 餘料退回 → 在「來源出貨單」選原本那筆 → 入庫位置自動標餘料。
5. 餘料累積到一定量 → 「出貨」→ 餘料轉撥出貨 → 選新工號 → 系統只扣餘料部分。
